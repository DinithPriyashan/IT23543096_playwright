## Running the Tests

From the `test\\\\\\\_automation` root directory, run:

```bash
python test\_automation.py --excel "IT23543096\_Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 10000 --type-delay-ms 100 --slow-mo-ms 500 --save-every 1 --keep-open     
          
```

### Command-Line Arguments

|Argument|Description|Default|
|-|-|-|
|`--excel`|Path to the Excel test cases file|Auto-detected|
|`--url`|URL of the application under test|`https://www.pixelssuite.com/chat-translator`|
|`--wait-ms`|Milliseconds to wait for output after input|`10000`|
|`--type-delay-ms`|Delay between keystrokes (ms)|`100`|
|`--slow-mo-ms`|Playwright slow-motion delay (ms)|`500`|
|`--save-every`|Save Excel every N rows|`1`|
|`--keep-open`|Keep browser open after tests finish|`false`|
|`--headless`|Run browser in headless mode (no UI)|`false`|
